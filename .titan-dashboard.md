# PhonkLeaderboard Dashboard

Welcome to the **PhonkLeaderboard** Dashboard.

## Project Vision
A specialized web platform where Funk/Phonk YouTube artists can paste their channel links to compete on a global leaderboard based on subscriber count (and eventually views/trending).

## Architecture & Blueprint (Locked)
Since you want a strong foundation and asked me to decide the defaults, I have locked in robust, scalable choices:
- **Framework:** Next.js (App Router) with TypeScript
- **Styling:** Tailwind CSS (using theme tokens and atmospheric dark styling that fits Phonk culture; strictly avoiding generic SaaS templates).
- **Icons:** `lucide-react` (strictly no emojis).
- **Database:** PostgreSQL via Prisma ORM (externalized so it can run flawlessly on Railway-like environments).
- **API integrations:** YouTube Data API for automatic channel stat synchronization.
